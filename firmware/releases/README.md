# PTV-TRMNL Firmware Releases v5.45

Pre-built firmware packages for all supported devices.

## Supported Devices

| Folder | Device | Resolution | Notes |
|--------|--------|------------|-------|
| `trmnl-og/` | TRMNL OG | 800×480 | ESP32-C3, e-ink |
| `kindle-pw5/` | Paperwhite 5 (11th gen) | 1236×1648 | USB-C, 6.8" |
| `kindle-pw4/` | Paperwhite 4 (10th gen) | 1072×1448 | Waterproof |
| `kindle-pw3/` | Paperwhite 3 (7th gen) | 1072×1448 | Classic PW |
| `kindle-11/` | Kindle 11th gen | 1072×1448 | Entry 300ppi |
| `kindle-basic-10/` | Kindle Basic 10th gen | 600×800 | 167ppi |

## Installation

### TRMNL OG
See `trmnl-og/FLASH-INSTRUCTIONS.md`

### Kindle Devices
1. Jailbreak with WinterBreak
2. Install KUAL
3. Copy device folder to `/mnt/us/extensions/ptv-trmnl/`
4. Open KUAL → PTV-TRMNL → Start Dashboard

## Server
All devices connect to: https://ptvtrmnl.vercel.app
